## 1. Successive Percentage Change

Jab kisi quantity par ek ke baad ek percentage change hota hai.

### Formula

```
Net % Change = a + b + (ab / 100)
```

> **Sign Rule:** Increase = `+` | Decrease = `−`

### All Cases

| Case | Example | Calculation | Result |
|------|---------|-------------|--------|
| Increase + Increase | +20%, +30% | 20 + 30 + (600/100) | **+56%** |
| Decrease + Decrease | −20%, −30% | −20 − 30 + (600/100) | **−44%** |
| Increase + Decrease | +30%, −20% | 30 − 20 + (−600/100) | **+4%** |
| Decrease + Increase | −40%, +20% | −40 + 20 + (−800/100) | **−28%** |

### 🔹 Example Question

> A shopkeeper marks up his price by 20% and then gives a discount of 10%. Find the net % change in price.

```
Net % = 20 + (−10) + (20 × −10)/100
      = 10 − 2
      = +8%  →  Net increase of 8%
```

## 2. Multiplication Factor Method

Har percentage ko ek factor me badal do, phir multiply karo.

### Factor Table

| Change | Factor |
|--------|--------|
| +10% | 1.10 |
| +20% | 1.20 |
| +25% | 1.25 |
| +50% | 1.50 |
| +100% | 2.00 |
| −10% | 0.90 |
| −20% | 0.80 |
| −25% | 0.75 |
| −40% | 0.60 |
| −50% | 0.50 |

### Formula

```
Final Factor = Product of all individual factors
Net % Change = (Final Factor − 1) × 100
```

### 🔹 Example Question

> A salary is increased by 20%, then by 10%. Find total % increase.

```
Factor = 1.20 × 1.10 = 1.32
Net % = (1.32 − 1) × 100 = 32%  →  Total increase = 32%
```

## 3. Repeated Increase

Jab same % se baar baar increase hota ho.

### Formula

```
Final = P × (1 + x/100)^n
```

### 🔹 Example Question

> Population of a city is 10,000. It grows at 10% per year. Population after 3 years?

```
Final = 10000 × (1.1)³
      = 10000 × 1.331
      = 13,310
```

---

## 4. Repeated Decrease

Jab same % se baar baar decrease hota ho.

### Formula

```
Final = P × (1 − x/100)^n
```

### 🔹 Example Question

> A machine worth ₹50,000 depreciates at 20% per year. Value after 2 years?

```
Final = 50000 × (0.8)²
      = 50000 × 0.64
      = ₹32,000
```

## 5. Reverse Percentage

Original value nikalni ho jab final value aur change % pata ho.

### Formula

| Situation | Formula |
|-----------|---------|
| After x% **increase** | `Original = Final ÷ (1 + x/100)` |
| After x% **decrease** | `Original = Final ÷ (1 − x/100)` |

### 🔹 Example Question

> After a 25% increase, a number becomes 500. Find the original number.

```
Original = 500 ÷ (1 + 25/100)
         = 500 ÷ 1.25
         = 400
```

---

## 6. Increase Required After Decrease

Decrease ke baad original value par wapas aane ke liye kitna % increase chahiye?

### Formula

```
Required Increase % = x / (100 − x) × 100
```

### 🔹 Example Question

> A price decreases by 20%. By what % must it be increased to restore original price?

```
= 20 / (100 − 20) × 100
= 20 / 80 × 100
= 25%
```

---

## 7. Decrease Required After Increase

Increase ke baad original value par wapas aane ke liye kitna % decrease chahiye?

### Formula

```
Required Decrease % = x / (100 + x) × 100
```

### 🔹 Example Question

> A price increases by 25%. By what % must it be decreased to restore original price?

```
= 25 / (100 + 25) × 100
= 25 / 125 × 100
= 20%
```

## 9. Percentage Change in Ratio

For ratio **A : B = m : n**

### Formulas

```
A is greater than B by  →  (m − n) / n × 100
A is less than B by     →  (n − m) / n × 100  (when m < n)
```

> ⚠️ **Rule:** Denominator hamesha us quantity ka hota hai jisse compare kar rahe ho.

### 🔹 Example Question

> Ratio of boys to girls = 5 : 4. By what % are boys more than girls?

```
= (5 − 4) / 4 × 100
= 1/4 × 100
= 25%  →  Boys are 25% more than girls
```

## 10. Population / Price / Salary Formulas

Sab same formula follow karte hain.

| Context | Formula |
|---------|---------|
| Population increase | `P × (1 + x/100)` |
| Price after decrease | `P × (1 − x/100)` |
| Salary after increment | `P × (1 + x/100)` |

### 🔹 Example Question

> A product costs ₹800. Price decreases by 15%. New price?

```
= 800 × (1 − 15/100)
= 800 × 0.85
= ₹680
```

## 11. Other Standard Formulas

### Percentage Difference

```
= |A − B| / Original × 100
```

> `Original` = jo value base hai question me.

**Example:** A = 80, B = 100. What % is A less than B?
```
= (100 − 80) / 100 × 100 = 20%
```

---

### Exam Marks Formula

```
Obtained Marks  =  (% / 100) × Total Marks
Total Marks     =  (Obtained × 100) / %
```

**Example:** A student scored 75% in exam of 200 marks. Marks obtained?
```
= (75/100) × 200 = 150 marks
```

---

### Attendance Formula

```
Present  =  (% / 100) × Total Students
Absent   =  Total − Present
```

**Example:** 80% of 500 students were present. How many absent?
```
Present = 400  |  Absent = 500 − 400 = 100
```

---

### Voter Questions

```
Votes received  =  (% / 100) × Total Votes
Invalid votes   =  Total − Valid Votes
```

**Example:** In an election of 10,000 votes, a candidate got 60% of valid votes. Valid votes = 9,000. Votes received?
```
= (60/100) × 9000 = 5,400 votes
```

## 12. Common SSC Trap ⚠️

```
❌  20% increase + 20% decrease = 0%   →  WRONG

✅  Correct:
    20 + (−20) + (20 × −20)/100
    = 0 − 4
    = −4%  →  Net DECREASE of 4%
```

> **Trap Question:** A number is increased by 20% and then decreased by 20%. Net change?
> **Answer: −4% (net decrease)**, not 0.

## 📋 Quick Formula Sheet

| Concept | Formula |
|---------|---------|
| Successive change | `a + b + ab/100` |
| Factor method | Multiply all factors, then `(result − 1) × 100` |
| Repeated increase | `P(1 + x/100)^n` |
| Repeated decrease | `P(1 − x/100)^n` |
| Reverse (after increase) | `Final ÷ (1 + x/100)` |
| Reverse (after decrease) | `Final ÷ (1 − x/100)` |
| Increase after x% decrease | `x/(100−x) × 100` |
| Decrease after x% increase | `x/(100+x) × 100` |
| Ratio comparison | `(Difference / Base) × 100` |
| 20% up + 20% down | `−4%` (not 0%) |
